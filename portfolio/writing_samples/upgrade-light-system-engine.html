Upgrading Light System Engine

Periodically, we update Light System Engine software to provide improvements, enhancements, and bug fixes. Up to two software versions can be installed on Light System Engine gen5 at any given time, so you can roll back an update if you encounter unexpected behavior.

Upgrade Light System Engine

Software updates are available for download at www.colorkinetics.com/support/lsm/. You can update your software through the LSE web interface.

To find your Light System Engine software version:
  1. Log in to the LSE web interface, and click Config.
  2. From the left menu, navigate to Settings > About. Your software version is the last entry on this screen.

If you're running version 2.0 or later, continue to To upgrade from version 2.0 or later. If you're running version 1.9.6 or earlier, contact Technical Support for upgrade instructions.

To upgrade from version 2.0 or later:
  1. Download the latest software file from www.colorkinetics.com/support/lsm/. Do not change the file name.
  2. If you already have two versions installed on Light System Engine, delete the inactive version:
    1. Log in to the LSE web interface, and click Config.
    2. From the left menu, navigate to Updates > Versions.
    3. Select the inactive version from the Current Version list, and click Delete. Light System Engine prevents you from deleting the active and recovery versions.
  3. Back up your Light System Engine database. If you need help, see Database Backup.
  4. Upload the software file from your computer to Light System Engine:
    1. In the Config page, select Updates > Upload from the left menu.
    2. Click Choose File, and then select the software file you downloaded in step 1.
    3. Click Save. Light System Engine shows the message that the software file was uploaded.
  5. Reboot Light System Engine.

All data is copied to the new software version. Data is stored independently with each software version, so any changes you make in the new version are not reflected in the previous version.

After rebooting, Light System Engine runs the new software version. If you can log in to the LSE web interface from your computer, you're done with the upgrade.

If Light System Engine fails to reboot, this means the upgrade wasn't successful. If this happens, you'll need to restore your Light System Engine to the previous software version. These instructions are shown in Recover From an Unsuccessful Upgrade.

Recover From an Unsuccessful Upgrade

Normally, Light System Engine is stored in a secure location without a keyboard or monitor. If you need to recover from an unsuccessful upgrade, you'll need a keyboard, a monitor, and physical access to Light System Engine.

To recover from an unsuccessful upgrade
  1. Turn off Light System Engine.
  2. Connect your keyboard and monitor, and then turn on Light System Engine. The boot loader screen is displayed.
  3. Using your keyboard arrow keys, select the previous software version, and then press Enter. You can now disconnect the keyboard and monitor.
  4. Delete the inactive software version:
    1. From your computer, log in to the LSE web interface, and click Config.
    2. From the left menu, navigate to Updates > Versions.
    3. Select the inactive version from the Current Version list, and click Delete.

If you are unable to boot into either software version installed on your Light System Engine, contact Technical Support.
