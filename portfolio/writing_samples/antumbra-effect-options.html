Lighting Effect Options

You can choose from six lighting effects in Antumbra iColor Keypad Effect Manager. Effects have different options for customization, such as duration, saturation, and color. See the table below for all the available options.

To set a lighting effect:
  1. Choose a keypad button from the left side of the window.
  2. Choose the lighting effect you want from the Effect list.
  3. Customize the effect, and click Save.

If you need help, this table shows each effect's options:
