Lighting Effect Options

You can choose from six effects in Antumbra iColor Keypad Effect Manager.

To set a lighting effect:
  1. Choose a keypad button from the left side of the window.
  2. Choose the lighting effect you want from the Effect list.
  3. Customize the effect, and click Save. If you need help, this table shows each effect's options:
    Effect Options
    Fixed Color
    All luminaires show the same color. Keypad dimming buttons adjust brightness.
    Color
    Select a color using the Color Picker
    Variable Color
    All luminaires show the same color. Keypad dimming buttons adjust hue.
    Color
    Select a color using the Color Picker
    Color Wash
    Smooth transition through the color spectrum. All luminaires show the same color.
    Duration
    Transition time, in m:ss
    Saturation
    Saturation value, from 0 (white) to 255 (fully saturated)
    Intensity
    Intensity value, from 0 (off) to 255 (full on)
    Direction
    Set colors to move forward or reverse through the color spectrum
    Rainbow Wash
    Smooth transition through the color spectrum. Colors follow from luminaire to luminaire.
    Duration
    Transition time, in m:ss
    Width
    Rainbow width, from 1 to 512 nodes
    Saturation
    Saturation value, from 0 (white) to 255 (fully saturated)
    Intensity
    Intensity value, from 0 (off) to 255 (full on)
    Direction
    Set colors to move forward or reverse through the color spectrum
    Custom Wash
    Smooth transition through 2 to 6 custom colors. Colors follow from luminaire to luminaire.
    Color Bar
    Add or delete effect colors, from 2 to 6 colors
    Color
    Select a color using the Color Picker
    Duration
    Transition time, in m:ss
    Width
    Rainbow width, from 1 to 512 nodes
    Direction
    Set colors to move forward or reverse through the color spectrum
    Random Color
    Random color shown on all luminaires. All luminaires show the same color.
    Duration
    Transition time, in m:ss
    Saturation
    Saturation value, from 0 (white) to 255 (fully saturated)
    Intensity
